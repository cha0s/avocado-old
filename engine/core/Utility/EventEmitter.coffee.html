Remove listeners from an object.

There are four ways to use avo.EventEmitter.off:

  1. You can call object.off 'eventName', function where function is a function previously attached with object.on eventName, function. If the function was never registered against this event, nothing happens.

  2. You can call object.off 'eventName.namespace' where namespace is a user-defined namespace. If no listener was registered under the current namespaced event, nothing happens.

  3. You can call object.off '.namespace' where namespace is a user-defined namespace. If no listener was registered under the current namespace, nothing happens.

  4. You can call object.off 'eventName'. This is generally undesirable, as Avocado registers event listeners against some built-in objects, and they can be easily be accidentally removed with this method. Use caution.
