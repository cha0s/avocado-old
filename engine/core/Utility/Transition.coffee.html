Transition.coffee

engine/core/Utility/
¶

avo.Transition is a Mixin which lends the ability to handle timed transitions of arbitrary property methods residing on the mixed-in object.

You can use this mixin like this:

avo.Mixin yourObject, avo.Transition

yourObject.transition {x: 100}, 2000, 'easeOutQuad'

The value of yourObject.x() will transition towards 100 over the course of 2000 milliseconds. NOTE: yourObject must have the functions x() and setX() defined.

This function was heavily inspired by the existence of jQuery.animate, though the API is NOT compatible.
